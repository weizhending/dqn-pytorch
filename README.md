# DQN PyTorch
This is a simple implementation of the Deep Q-learning algorithm on the Atari Pong environment.

![](/assets/pong.gif)

System dependencies for gym environment:
* cmake 
* libz-dev
* (build-essential)
* (manpages-dev)

Python depedencies:
* gym
* gym[atari]
* opencv-python
* torch
* torchvision

## References
1. *Playing Atari with Deep Reinforcement Learning*, Mnih et al., 2013
